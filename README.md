## Tecnologias
Se utilizo Vue3 con el compilador Vite en conjunto con Typescript

## Instalacion
- Clonar el repositorio
- Instalar nodejs con npm
- Correr
```
npm i
npm run dev
```


## Comandos
- `npm i` instala las dependencias de npm necesarias
- `npm run dev` corre el servidor
